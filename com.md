cargo run --bin maazdb-server    

Run this command to install it globally:

```bash
cargo install --path /run/media/maaz/S/project_maazdb/
```

Now you can just type **`maazdb-cli`** from any folder.

cargo run --bin maazdb-cli monitor

cargo run -- -c

cargo run -- --c

cargo run -- -config

cargo run -- --config

cargo run -- c

cargo run -- config

cargo run --bin simple-test  

cargo run --bin maazdb-test
   
cargo build

cargo build --release

cargo run --bin maazdb-bench-v4 -- --type all

cargo run --release --bin maazdb-server
